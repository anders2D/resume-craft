# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Multi-language support (Spanish/English)
- Advanced page setup with custom dimensions
- Print optimization with zoom control
- AI-powered CV optimization
- Inline editing for all content
- Import/Export CV data as JSON
- Experience management (add, reorder, delete)
- Skills editor with categories
- ATS optimization guidelines

### Changed
- Improved print layout and styling
- Enhanced responsive design
- Centralized CRUD operations with useCVData hook

### Fixed
- Data persistence issues with ProfileContext
- Job title and company display on print
- "Present" translation to "Actualidad" in Spanish

## [0.1.0] - 2025-01-XX

### Added
- Initial release
